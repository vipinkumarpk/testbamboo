FMCUser
=======
By Finbarr McCarthy
**What is FMCUser?**

FMCUser is a Module that acts like glue between ZfcUser, ZfcUserDoctrineORM and BjyAuthorize.

Dependencies
============

This Module heavily depends on the following Modules and makes no sense without them:

 - ZfcUser
 - DoctrineORMModule
 - ZfcUserDoctrineORM
 - BjyAuthorize
